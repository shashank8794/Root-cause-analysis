Steps involved to perform root cause analysis:-
1. create a virtual environment and install all the packages listed    in requirement.txt file
2. activate the virtual environment
3. go to the directory where all the files are located. Here, go to /code    directory.
3. keep the failure dataset excel file in dataset/excel folder
4. For data preprocessing, either you can run data_preprocessing_v7.py or you can    do it using the interface. 
	Run data_preprocessing.py . It will generate 2 pickle files in          dataset/pickle folder
5. Run app.py
       a link will get generated. Open this link in your browser and start using        the interface as per your requirement.
